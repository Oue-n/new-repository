-実行方法
　AIWOLFフォルダ内のAutoStarter.batを実行することで人狼の対戦が始まります。
  (Macの場合AutoStarter.shを実行)
　(環境としてJavaとPythonのファイルが動作することが必要です。)
  AutoStarter.iniで設定を変更できます。
　view = false とするとコンソール上で高速に動作します。
　複数回の試合における勝率を出したいときはこちらが推奨されます。
　view = true とするとGUIが起動し視覚的に対戦内容が把握できます。
　その他の設定は以下の公式HPをご覧ください。
　http://aiwolf.org/server
